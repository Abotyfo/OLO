# OLO Specification
Os Termos de Uso podem ser séria, então se você não sabe sobre a OLO, vê esse texto
A primeira coisa importante é que se quiserem copiar a linguagem, usa essas recomendações:
- Não use código difícil sem ter explicação por "uso" ou "sintaxe própria"
- Se ter engano, não recomendo corrigir sozinho, senão pode piorar; Peça para um adulto ou um programador profissional para corrigir com segurança
- Se tiver "uso", não use pelo "público"; Hackers podem roubar o código e te mandar coisas ilegais, se tiver dúvidas, não há.
