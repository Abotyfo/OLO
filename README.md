# OLO
OLO é uma esolang feita para desafiar os gênios e as IAs. Consegue programar ou aprender?
## Exemplo de sintaxe
```olo
A
>÷*%[@[&+[$>^*#(^#[×>/*%)&]#<×>[=<:*,"*@[=>*$>;][@&@*$[:$*&#[/*:*@&@[>/*@[+$*%*'[@>%[",:,-(@(*%]$[!&=[">;
Z
```
Provavelmente esse código é difícil, caso tenha ajuda, é só estudar.
## Como fazer
Tudo é rodado entre `A` e `Z`
Entre essas letras, entra o caos. Os caracteres tem em especial, mas pode ser bem difícil, né?
## Atalho
Se juntar `@` e `<>`, você cria uma repulsa. Mas o que é repulsa? A repulsa é **aquilo que vai ser executado e julgar o outro para não ser julgado**
## Como seguir o ponto
Estude bem e não segue a tal 'moda da *internet*', segue o estudo, assim aprende OLO!
